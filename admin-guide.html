Admin Panel Guide

Complete guide to managing your Gluten Free India website

1. Accessing the Admin Panel

The admin panel is built into the products page for easy access. Here's how to use it:

  1. Go to the Products page on your website
  2. Look for the "Admin Panel" button in the filters section
  3. Click on it to reveal the product management interface
  4. Use "Hide Admin" to close the panel when done

2. Adding New Products

To add a new product to your website:

  1. Product Name: Enter the full name of your product
  2. Category:
    • Select from existing categories (Flours, Breads, Cakes, etc.)
    • OR create a new category by typing in the "new category" field
  3. Price: Enter the price in rupees (numbers only)
  4. Image URL:
    • Upload your image to a hosting service (like Imgur, Google Drive, etc.)
    • Copy the direct image URL and paste it here
    • Leave blank to use a placeholder image
  5. Description: Write a detailed description of the product
  6. Click "Add Product" to save

3. Managing Categories

Existing Categories:

  • Flours & Healthy Mixes
  • Breads & Fresh Bakery
  • Cakes & Muffins
  • Cookies & Biscuits
  • Snacks
  • Breakfast Items

Creating New Categories:

Simply type a new category name in the "new category" field when adding a product. The system will automatically create it and add it to the dropdown menus.

4. Deleting Products

To remove a product:

  1. Find the product in the products grid
  2. Click the red "Delete" button on the product card
  3. Confirm the deletion in the popup dialog

Currently, all product data is stored in your browser's local storage. This means:

  • Data persists only on the device/browser where you add it
  • Clearing browser data will delete all products
  • Other users won't see products added from your device

For a production website, you'll need to implement a proper database backend.

6. Image Management

Best Practices for Product Images:

  • Size: Recommended 400x300 pixels or similar ratio
  • Format: JPG or PNG
  • Quality: High quality but optimized for web
  • Hosting: Use reliable image hosting services

Free Image Hosting Options:

  • Imgur: imgur.com - Easy to use, direct links
  • Google Drive: Share images publicly and get direct links
  • Cloudinary: cloudinary.com - Professional image hosting

7. Website Deployment

For Production Use:

  1. Upload Files: Upload all website files to your web hosting server
  2. Domain Setup: Point your domain to the hosting server
  3. SSL Certificate: Enable HTTPS for security
  4. Database Setup: Replace localStorage with a proper database

Recommended Hosting Platforms:

  • Netlify: Free hosting for static sites
  • Vercel: Great for modern web apps
  • GitHub Pages: Free hosting via GitHub
  • Traditional Hosting: Hostinger, Bluehost, etc.

8. Upgrading to a Database

For a production website, you'll want to replace localStorage with a real database:

Backend Options:

  • Node.js + MongoDB: Popular JavaScript stack
  • PHP + MySQL: Traditional web development
  • Firebase: Google's backend-as-a-service
  • Supabase: Open-source Firebase alternative

10. Troubleshooting

Common Issues:

  • Images not loading: Check if the image URL is accessible and direct
  • Products disappearing: Browser data might have been cleared
  • Categories not showing: Refresh the page after adding new categories
  • Form not submitting: Ensure all required fields are filled
